### Branching

main - This branch will hold our releases
<br/><br/>
develop - This will contain our working project when features and bugs are fixed and PR'ed
<br/><br/>
feature/[featureName] = This branch will contain everything from this feature that we are developing
<br/><br/>
hotfix/[name] - This branch will automaticallt go into production this branch type will only be used when there is a production critical bug
